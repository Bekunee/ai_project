# ai_project (Backend)
This is the backend API for a market application for visually impaired users.  
Built with **FastAPI + MongoDB**.

## Features:
- 📦 Get all products (with pagination)
- 🔍 Search products by name
- 🆔 Get product by ID
- 🛒 Record product purchase
